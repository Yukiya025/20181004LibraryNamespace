# importを使ったライブラリの呼び出し方法集
ライブラリの呼び出し方法は3つあります。
1. import [ライブラリ名]
2. from [ライブラリ名] import [関数名]
3. from [ライブラリ名] import *

このレポジトリでは、上記3つの方法を使いわけるためのサンプルコードを5つとインポート用ライブラリを1つ用意しました。
1. main.py
2. main1.py
3. main2.py
4. main3.py
5. main4.py
6. wordifier.py (インポート用ライブラリ)
